# CC 选课指南

### O、什么是 CC Course

CC 的全称为 Common Core，所有课程被分成了五个不同的领域，分别是：Scientific and Technological Literacy（课号简称 CCST），Arts and Humanities（课号简称 CCHU）， Global Issues（课号简称 CCGL）， China: Culture, State and Society（课号简称 CCCH），与于2025-2026学年新开设的Artificial Intelligence（课号简称CCAI）。港大创立 CC 课程的初心是鼓励大家探索自己的兴趣爱好，帮助大家与自己的未来建立关系。更多信息请进入下一部分中的官方网站查看。

<mark style="color:blue;">需要选几门 CC?</mark>

对于新入学的四年制本科项目，大部分专业的同学们需要在四年内修够六门 CC（每个专业具体需要修够的数量以自己专业的 Syllabus 要求为准）。其中 CH/HU/ST/GL/AI 每个类别至少一门，至多两门。同学们可以根据学院的要求自由地安排每学年CC的数量，但由于每个人手速/网速不同，很有可能不能完全选上首选的 CC，所以我们建议大家多准备一点不太热门的备选 CC。

<mark style="color:blue;">一年内可以选两门相同类别的 CC 吗?</mark>

只要确保四年内该类别课程不超过两门，一年内 CC 总课程不超过四门，也就是说，同一年乃至同一学期内 enroll 两门 CH/HU/ST/GL/AI 都是被允许的～

### 二、怎样抢到课

<mark style="color:blue;">CC Enroll 的规则?</mark>\
正常选课阶段是先到先得, add\&drop period 是定时抽签。如何在正常选课阶段快速选课可以参考推送 [HKU 选课系统使用指南](https://mp.weixin.qq.com/s/D_yThNRc8FMX-lIfGNo6kw)。\
PS: Full-time 的同学需要选择不带有"X"的 subclass，因为带"X"的是仅供交换的同学选择的。

<mark style="color:blue;">选课时如何知道自己有没有抢到 CC?</mark>\
同学可在 Portal 中 Common Core Enrollment Status 查看自己所在的位置，如果自己的位置数低于剩余名额，则表示已经成功选上。而如果位置数比剩余名额高，则很可能会选不上（除非排在自己前面的选上的人 drop 了这门课程）。一旦有可能选不上，同学需要尽快决定是要放弃等待这门课程转而选择备选课程（因为如果等待更长时间，备选课程可能也会被抢完），还是继续等待 add\&drop period。

<mark style="color:blue;">在 add\&drop period 抢 CC 课需要时时蹲点吗?</mark>\
不需要!\
好消息来啦！不要再深夜蹲守、课上心神不宁地刷 portal 了！卡准时间点的话，每天二十分钟就够了～\
如上题所述，在 add\&drop period 内抽签的结果与 add 的先后顺序无关，因此只要在每一次抽签之前 add 上仍然有 Quota 的 CC 课就可以。那么什么时候抽签呢？根据官方历史公布的数据，抽签的具体时间如下。\
Suspension Period 一次;\
Add\&Drop Period 中：每天的 8am, 11am, 2pm, 5pm, 8pm, 11pm 各一次。\
因此最经济省时间的做法，就是在这六个时间点前查看这门 CC 是否有 quota，\
及时 Add，并在相应时间点后检查自己的 Enrollment status，如果是 Not Approved，记得及时把这门课重新加入 Temporary List哟。

### 怎么知道某门课还有没有 Quota 呢?

一般有如下两种方法:

#### a. 最实时：

把那门课加入你的 temporary course list，每次刷新就可以看到 status 上是 closed（蓝色小方块）还是 open（绿色小圆圈）。一旦出现 open，就可以赶紧 Proceed to Step 2 of 3\
（前提是总课程数小于等于六门，否则需要向学院申请 Overload）啦！

#### b. 最方便:

学校有 common core course enrolment statistics 可供查看，可以点击 RIC 杂货铺"选课平台"中的"选课实时数据"或者复制以下链接至浏览器就能看到啦～

[https://sweb.hku.hk/ccacad/ccc\_appl/enrol\_stat.html](https://sweb.hku.hk/ccacad/ccc_appl/enrol_stat.html)\
这个地址的好处是不仅能看到 open 的状态, 还可以看到空缺几个名额, 有多少人在 queue up; 坏处是这个网站每小时定时更新一次, 所以上面的数据可能会有延迟。

### 三、Common Core Clusters and Transdisciplinary Minor

Common Core Thematic Cluster 是一系列相同主题的 CC 课程。如果在同一 Cluster 下选择四门 CC 课, 这个 Cluster 将会被记录在 Academic Attainment Profile (AAP)上; 如果在同一 Cluster 下选择 6 门 CC 课, 则可以构成一个 Transdisciplinary Minor。（如果选的课同时满足多个 Cluster, 只能选择其中一个记录在 AAP 上)\
如果已经 declare 了一个 Common Core Transdisciplinary Minor（CCTM），在选剩余的 CC课时可以享受优先权（CCTM 在选择四门同一 Cluster 的 CC 课后才可以 declare)

所有 Thematic Cluster 所包含的课程列表可以打开以下链接查看:\
[https://commoncore.hku.hk/thematic-clusters/](https://commoncore.hku.hk/thematic-clusters/)\
详细政策可以在打开以下链接查看:\
[https://commoncore.hku.hk/cctms/](https://commoncore.hku.hk/cctms/)

### 四、其他问题

**问：能不能把 CC 课上成 free elective / 我能不能上超过六门 CC?**\
答: 很遗憾, 这是学校绝对不允许的。四年六门的限制是一旦选满, 绝不可以通过任何形式申请 Overload 或者转成 Free 学分的。\
所以大家要谨慎选择心爱的 CC, 多看多听（可以在 RIC 杂货铺的选课平台中查看大家对 CC 课的评价～），假如实在喜欢，也可以选择 Sit In。

**问：选不上课，能不能写邮件 argue?**\
和专业课程不同, CC 课的 Enroll 为了确保绝对的公平公正, 课程相关的 Lecturer 和 Tutor是不可以参与到选课过程中来的。

**特殊规则：对于 17 年之后入学的同学，有对 Graduation GPA 的特殊规则**：

GGPA 将有两种计算方案：GPA 最高的五门 CC 课（需要 cover 全部四种课程）或者全部六门 CC 课, 最终将以高的方式计算。\
详情可以复制以下链接至浏览器查看 CCC 官网关于此项目的详细情况\
[https://commoncore.hku.hk/special-proviso/](https://commoncore.hku.hk/special-proviso/)\
更多详细信息大家也可以复制以下链接至浏览器查看:\
[http://commoncore.hku.hk/faqs/](http://commoncore.hku.hk/faqs/)
