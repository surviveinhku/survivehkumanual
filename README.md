# README

RIC内测版本，先将新生群文件洗稿为markdown。

现在的问题是pdf转markdown的开源脚本大多不兼容图片和复杂格式，兼容图片的又得付费，所以我现在给[https://pdf-to-markdown.com](https://pdf-to-markdown.com/)氪金了五美刀，足够转1000页，效果实测下来不错，每个文档需要proofreading+重新上超链接五分钟左右。
