# 关于生存手册

> 他（歷史天使）面向過去。在我們看來是一連串事件的地方，他看到的是一場單一的災難，不斷地把殘骸堆疊在殘骸之上，並將它們擲到他腳前。\
> 他想要停下來，喚醒死者，將破碎之物重新拼湊起來。\
> 但有一股來自天堂的風暴吹來，扯住了他的翅膀，強大到讓他無法將翅膀收回。\
> 這股風暴無情地將他推向未來，然而他背對著未來，被迫向前飛去，眼前的廢墟堆積得越來越高，直衝天際。\
> 這場風暴就是我們所指的「進步」（Fortschritt）

### 这是一份怎样的手册？

这不仅仅是一个教你如何拿满绩，刷实习，成为中环精英的手册。<br>

RIC的庄友们汇总了过去的新生群文件与《如何优雅地XXX》系列作为手册的骨架，而需要你们的真实感想去构筑《香港大学生存手册》的血肉。

作为港大的内地生群体，我们虽无法代表任何人，但我们所身处的环境是相同的：

* 在亲戚朋友的眼里，我们看到成为“大人和大人物”的期许，看到了西装革履与对于“可能性”的无限幻想。
* 在 X 与 Threads 上，我们看到许多人对内地生“nerd”“做题家”的刻板印象；
* 在大大小小的庄里，我们看到许多上庄的同学逐渐承担起对外表达的角色，在不同场合中使用更正式、规范的语言与方式。

但这些都不是我们真实生活的全貌。

在这座以效率与原子化著称的城市里，当你身处夜深人静，面对着涣散的未来……\
那些关于自己、关于港大的真实念头，是否会因为觉得“无人会在意”，而悄悄自我消解？

**我们想听你的故事。**

### 这是一个面向 **B1–B28** 的长期征稿计划。<br>

从 **2026 年起**，我们征集每一个港大内地生的文章投稿与线上采访：

一方面在RIC旗下公众号《港大浮世绘》上发布，并在RIC小红书账号上同步转发。

另一方面精选其中文章收录至本Gitbook网站，用一年的时间汇聚大家的短篇。

我们希望它不只呈现少数被频繁书写的经验，

而是一幅能让更多来到港大的内地学子产生共鸣的浮世绘。

### 关于HK

在香江对岸，作为大陆学生的你还适应吗？有哪些瞬间让你觉得你是异乡人，又有哪些时刻让你感受到了归属感与自由？你怎么交到的第一个local朋友？在狮子山下，异乡人的失意或成长，都值得被听见。

### 关于学业

在HKU你还满意吗？是否有时会后悔没有去读清北，华五，牛剑，美本，付着高额的学费来到港大？在时隔X年后，如果给你一个重新开始的机会，你会如何选择？

### 我们欢迎谁的投稿？

#### 如果你在香港大学度过了成功的四 / 五年，想分享自己一路的艰辛与信念：

我们欢迎你的投稿。\
或许你的故事会感染此时此刻在港大陷入迷茫的年轻人，让狮子山精神代代传承。

> “同舟人，世相随，无畏更无惧。”

#### 如果你自认为是“沉默的大多数”，认为自己的观点不被普世价值所容：

我们欢迎你的投稿。\
在遵守平台基础发布规则的前提下，我们承诺不修改、不删减。\
您的观点，值得被原原本本地听见。

> “既是同舟，在狮子山下且共济，抛弃区分求共对。”

#### 如果你是亚文化的爱好者，担心我们会驳回“不严肃的文章”：

我们欢迎你的投稿。\
一切亚文化皆为时代之症候，让我们“斜目而视”，看穿现实的矛盾与张力。

#### 如果你不愿被别人知道你的真实身份：

我们欢迎你的投稿，尊重每一份匿名的针砭时弊。



我们收录成功，也收录失败。收录中环精英的标准路径，也收录web3的投资者的未来展望。收录4.3满绩的神话，也收录"挂科后我发现世界没有塌"的幸存者偏差。
